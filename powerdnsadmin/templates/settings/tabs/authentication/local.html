Local Authentication Settings

Local User Login

This determines whether users can authenticate with accounts in the local PDA database.

Default Value: Enabled

You should ensure that you have appropriate admin rights assigned to at least one external auth provider account!

Local User Open Registration

This determines whether users are given the option to self-register at the login screen.

Default Value: Enabled

It should be understood that access to this application should be properly restricted to prevent wide open access to the Internet!

Password Complexity Policy

This determines if the following password complexity restrictions should be applied when creating, registering, or updating an account.

Default: Disabled

You may experience an increase in the amount of user complaints received as some people might find this annoying.

It is highly recommended that you enforce more secure password requirements on your users!

This defines the minimum complexity score required for a user password. The complexity score is calculated using the zxcvbn library. The default value of the log factor is 11 as it is considered secure. More information about this can be found here.

Default Value: 11

Password Character Policy

This determines whether the following password character restrictions should be applied when creating, registering, or updating an account.

Default: Disabled

If this is not enabled, it may lead to users making really poor, and often uninformed decisions about the quality of their chosen passwords.

This defines the minimum number of characters required for a user password.

Default: 10

This defines the minimum number of numeric characters required for a user password.

Default: 2

This defines the minimum number of lowercase characters required for a user password.

Default: 3

This defines the minimum number of uppercase characters required for a user password.

Default: 3

This defines the minimum number of special characters required for a user password.

Default: 1

Special characters include:

`!@#$%^&*()_-=+[]\{}|;:",.></?.
